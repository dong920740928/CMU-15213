大学时期我用csapp第二版的[web site](http://csapp.cs.cmu.edu/2e/students.html)，并完成了配套的lab，但第二版是基于ia32的。
后来第三版的英文版终于在大陆上架，其基于x64翻新了内容，并且[web site](http://csapp.cs.cmu.edu/3e/home.html)也提供了新版的lab(主要是attack和architecture)，
因此一段重新刷lab的旅程就此开始。
